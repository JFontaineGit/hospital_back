## API Reference

#### Redirect to frontend

```http
  GET /
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `None` | `None` | Not need token |

#### Get item

```http
  GET /{token}/api/{path}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `authorization`      | `string` | **Required**. token to fetch |
